# Overview
 A package template

## Prerequisites
- React

## Quickstart
- npm start
- npm link
- switch to project
- npm link <package_name>

## Deployment
- npm version [major|minor|patch]
- npm run build
- npm publish
- npm run git

## Setup Husky
- npm run husky
- create .husky/pre-commit
- add `npm run clean`
